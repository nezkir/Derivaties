Worked Examples

Example 1 The ε–δ Proof
Prove using the ε–δ definition that limx→1(5x − 3) = 2, and find δ explicitly in terms of ε.
1
State the goal
We need to show: given any ε > 0, we can find δ > 0 so that whenever 0 < |x − 1| < δ, it follows that |(5x − 3) − 2| < ε.
2
Simplify the expression
|(5x − 3) − 2| = |5x − 5| = 5|x − 1|
3
Work backwards to find δ
We need 5|x − 1| < ε, which means |x − 1| < ε/5. So we choose δ = ε/5.
4
Verify the choice
If 0 < |x − 1| < δ = ε/5, then |(5x − 3) − 2| = 5|x − 1| < 5 · (ε/5) = ε. ✓
✓   δ = ε/5 works. The limit limx→1(5x − 3) = 2 is proved.
Example 2 Full Curve Sketch
Sketch f(x) = (x + 1)² / (1 + x²), identifying all extrema, inflection points, and asymptotes.
1
Domain and Symmetry
Domain: all real numbers. Testing f(−x) gives neither f(x) nor −f(x), so there is no axis or origin symmetry.
2
First Derivative — find critical points
Applying the quotient rule and factoring the numerator:
f ′(x) = 2(1 − x²) / (1 + x²)²
Setting the numerator to zero: 1 − x² = 0, so x = −1 and x = 1 are the critical points.
3
Sign chart for f ′ — increasing/decreasing
On (−∞, −1): f ′ < 0, so f is decreasing.
On (−1, 1): f ′ > 0, so f is increasing.
On (1, ∞): f ′ < 0, so f is decreasing.

Therefore: local minimum at x = −1, where f(−1) = 0, and local maximum at x = 1, where f(1) = 2.
4
Second Derivative — concavity and inflection points
f ″(x) = 4x(x² − 3) / (1 + x²)³
Setting f ″ = 0 gives x = 0 and x = ±√3. Check sign changes:

(−∞, −√3): concave down.   (−√3, 0): concave up.
(0, √3): concave down.   (√3, ∞): concave up.

Inflection points at x = 0, x = −√3, and x = √3.
5
Asymptotes
Divide numerator and denominator by x² as x → ±∞: the function approaches 1. Therefore y = 1 is a horizontal asymptote. The denominator 1 + x² is never zero, so there are no vertical asymptotes.
Range = [0, 2].   Absolute min = 0 at x = −1.   Absolute max = 2 at x = 1.   Horizontal asymptote: y = 1.
Example 3 Integration by Parts
Evaluate ∫ x cos x dx.
1
Choose u and dv using LIATE
LIATE order: Logarithmic → Inverse trig → Algebraic → Trigonometric → Exponential. Algebraic comes before Trigonometric, so set u = x and dv = cos x dx.
2
Differentiate u and integrate dv
du = dx     and     v = sin x
3
Apply the formula ∫ u dv = uv − ∫ v du
∫ x cos x dx = x sin x − ∫ sin x dx
4
Evaluate the remaining integral
∫ sin x dx = −cos x, so the expression becomes x sin x − (−cos x) + C.
∫ x cos x dx = x sin x + cos x + C
Example 4 Volume by the Washer Method
The region bounded by y = x² + 1 and y = −x + 3 is revolved about the x-axis. Find the volume of the resulting solid.
1
Find the intersection points
Set x² + 1 = −x + 3. Rearranging: x² + x − 2 = 0. Factoring: (x + 2)(x − 1) = 0. Intersections at x = −2 and x = 1.
2
Identify the outer and inner radii
On [−2, 1] the line y = −x + 3 lies above the parabola y = x² + 1. So:
Outer radius R(x) = −x + 3
Inner radius r(x) = x² + 1
3
Set up the washer integral
V = π ∫−21 { (−x + 3)² − (x² + 1)² } dx
4
Expand the integrand
(−x + 3)² = x² − 6x + 9
(x² + 1)² = x⁴ + 2x² + 1
Difference = −x⁴ − x² − 6x + 8
5
Integrate and evaluate
∫ (−x⁴ − x² − 6x + 8) dx = −x⁵/5 − x³/3 − 3x² + 8x
Evaluating from x = −2 to x = 1 gives the value 117/5.
V = 117π/5
Example 5 Improper Integral
Evaluate ∫0∞ dx / (1 + x²). Determine whether it converges or diverges.
1
Convert to a limit — never plug in ∞ directly
∫0∞ dx/(1 + x²) = limt → ∞ ∫0t dx/(1 + x²)
2
Apply the standard integral formula
= limt → ∞ [ tan−1(x) ]0t = limt → ∞ [ tan−1(t) − tan−1(0) ]
3
Evaluate the limit
As t → ∞, tan−1(t) → π/2. Also, tan−1(0) = 0. Therefore the limit = π/2 − 0 = π/2.
∫0∞ dx / (1 + x²) = π/2.   The integral converges.

Common Pitfalls

Functions and Limits
⚠ Checking only one side of a limit
For piecewise functions and floor/ceiling functions, always verify that limx→a+ f(x) = limx→a− f(x) before concluding the two-sided limit exists. A single-sided check is never enough.
⚠ Claiming even or odd symmetry without algebra
Never rely on a graph alone. You must show f(−x) = f(x) for even, or f(−x) = −f(x) for odd, for every x in the domain. Functions like x² + x are neither — they fail both tests.
⚠ Forgetting that composite functions are not commutative
(f ∘ g)(x) ≠ (g ∘ f)(x) in general. Always substitute carefully, simplify fully, and check the domain of the result separately.
Curve Sketching and Derivatives
⚠ Treating f ″(c) = 0 as sufficient for an inflection point
f ″(c) = 0 is a necessary condition, not sufficient. You must confirm that f ″ actually changes sign at c. Classic counter-example: f(x) = x⁴ has f ″(0) = 0 but the concavity does not change — there is no inflection point at x = 0.
⚠ Misidentifying stationary points as extrema
If f ′ does not change sign at c — for example, f ′ goes from + to 0 to + — there is no local extremum at c. It is just a stationary (flat) point.
⚠ Ignoring endpoints when finding absolute extrema
On a closed interval [a, b] you must evaluate f(a) and f(b) as well as all interior critical points. The absolute maximum or minimum can occur at an endpoint, not just in the interior.
Integration
⚠ Wrong choice of u in integration by parts
Follow LIATE: Logarithmic → Inverse trig → Algebraic → Trigonometric → Exponential. Choose u as the type appearing earliest. The wrong choice leads to a harder remaining integral, not a simpler one.
⚠ Omitting the constant of integration
Every indefinite integral must end with "+ C". Forgetting it is an automatic mark deduction and signals a misunderstanding of the family of antiderivatives.
⚠ Plugging ∞ directly into an improper integral
You must always write the limit first: ∫a∞ f(x) dx = limt→∞ ∫at f(x) dx. Evaluate the definite integral symbolically first, then take the limit. Skipping this step is a conceptual error.
⚠ Missing Type II improper integrals hidden inside a finite interval
An integral like ∫04 1/(x − 3) dx is improper at x = 3, even though both bounds are finite numbers. Always scan the integrand for vertical asymptotes inside (or at the edges of) [a, b].
⚠ Swapping the outer and inner radii in the washer method
Always sketch the region first. R(x) is the distance from the axis of revolution to the outer curve. r(x) is the distance to the inner curve. Swapping them produces a negative integrand and a wrong answer.
Last-Minute Test Tips
✓Read every problem carefully before writing anything. Identify the technique required first.
✓For curve sketching, always follow the seven steps: domain → f ′ → critical points → f ″ → concavity → asymptotes → sketch.
✓State your final answer clearly, with the correct sign, and include + C for every indefinite integral.
✓For improper integrals, explicitly state whether the integral converges or diverges after evaluating the limit.
✓After partial fractions, substitute a convenient value of x back to verify your constants.
✓In washer-method problems, always confirm which curve is farther from the axis before assigning R and r.
