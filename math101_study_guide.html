Applications of Derivatives
Theorem / ToolStatement / Formula
1st Derivative Test f′ changes − to + at c ⇒ local min; f′ changes + to − at c ⇒ local max; no sign change ⇒ no extremum
2nd Derivative Test f′(c) = 0 and f″(c) < 0 ⇒ local max; f′(c) = 0 and f″(c) > 0 ⇒ local min; f″(c) = 0 ⇒ inconclusive
Critical point Interior point where f′ = 0 or f′ is undefined
Absolute extrema method (1) Evaluate f at all critical pts; (2) evaluate at endpoints; (3) largest = max, smallest = min
Concave up f″ > 0 on I; equivalently f′ is increasing on I
Concave down f″ < 0 on I; equivalently f′ is decreasing on I
Point of inflection f″ changes sign at that point (concavity changes)
Rolle's Theorem f continuous on [a,b], differentiable on (a,b), f(a) = f(b) ⇒ ∃ c ∈ (a,b): f′(c) = 0
Mean Value Theorem f continuous on [a,b], differentiable on (a,b) ⇒ ∃ c: f′(c) = (f(b)−f(a))/(b−a)
Linearization L(x) = f(a) + f′(a)(x−a)
Important linearization (1+x)k ≈ 1+kx (for x near 0)
Horizontal asymptote y = b if limx→±∞ f(x) = b
Vertical asymptote x = a if limx→a± f(x) = ±∞
Oblique asymptote Divide numerator by denominator; remainder → 0 as x → ±∞; asymptote is the linear quotient part

Worked Examples

1
ε-δ Verification of a Limit
Evaluate limx→1 (5x − 3) and verify the result using the ε-δ definition.
1
Find the Limit
Simply substitute x = 1 into the expression (it's a polynomial, so direct substitution works).
limx→1(5x − 3) = 5(1) − 3 = 5 − 3 = 2
2
Set up the Verification
We must show: for every ε > 0, there exists δ > 0 such that if |x − 1| < δ, then |(5x − 3) − 2| < ε. Work backwards from the conclusion.
|(5x − 3) − 2| < ε
|5x − 5| < ε
5|x − 1| < ε
|x − 1| < ε/5
3
Choose δ
The working shows that if |x − 1| < ε/5, the condition is satisfied. So we choose δ = ε/5.
Let δ = ε/5
4
Write the Proof
Now verify forward: given |x − 1| < δ = ε/5, we show the limit condition holds.
|(5x − 3) − 2| = |5x − 5| = 5|x − 1| < 5 · (ε/5) = ε ✓
✓ Final Answer
limx→1(5x − 3) = 2. Verified: choose δ = ε/5; then |x−1| < δ ⇒ |(5x−3)−2| = 5|x−1| < 5·(ε/5) = ε.
2
Limit Using Standard Series / Substitution
Evaluate: (A) limx→0 (esin x − 1)/x    (B) limx→0 x(ex − 1)/(1 − cos x)
A1
Part A — Multiply and Divide by sin x
The expression has esin x − 1 in the numerator. We know lim (eu−1)/u = 1 when u→0. Make sin x the "inner" variable.
limx→0 (esin x − 1)/x = limx→0 [(esin x − 1)/sin x] · [sin x / x]
A2
Part A — Apply Both Standard Limits
As x→0, sin x→0, so (esin x − 1)/sin x → 1. Also (sin x)/x → 1.
= 1 × 1 = 1
B1
Part B — Use 1 − cos x = 2 sin2(x/2)
Replace 1 − cos x with 2 sin2(x/2), then rearrange to isolate standard limits.
limx→0 x(ex−1)/(1−cos x) = limx→0 [(ex−1)/x] · [x2 / (2 sin2(x/2))] · 2
B2
Part B — Evaluate Each Piece
(ex−1)/x → 1; (x/2 / sin(x/2))2 → 1 since sin θ/θ → 1.
= 1 × 1 × 2 = 2
✓ Final Answers
(A) limx→0 (esin x−1)/x = 1  |  (B) limx→0 x(ex−1)/(1−cos x) = 2
3
Continuous but Not Differentiable
Show that f(x) = xe1/x/(1+e1/x) for x ≠ 0 and f(0) = 0 is continuous but not differentiable at x = 0.
1
Check Continuity — Right-Hand Limit
As x → 0+, 1/x → +∞, so e1/x → +∞. Divide numerator and denominator of f(x)/1 by e1/x.
limx→0+ f(x) = limx→0+ x / (e−1/x + 1) = 0/(0+1) = 0
2
Check Continuity — Left-Hand Limit
As x → 0−, 1/x → −∞, so e1/x → 0.
limx→0− f(x) = limx→0− xe1/x/(1+e1/x) = 0·0/(1+0) = 0
3
Conclusion on Continuity
LHL = RHL = f(0) = 0, so f is continuous at x = 0. ✓
LHL = RHL = f(0) = 0 ⇒ continuous at x = 0
4
Check Right-Hand Derivative
Compute [f(x)−f(0)]/(x−0) as x → 0+.
limx→0+ f(x)/x = limx→0+ e1/x/(1+e1/x) = 1 (since e1/x → ∞)
5
Check Left-Hand Derivative
As x → 0−, e1/x → 0.
limx→0− f(x)/x = limx→0− e1/x/(1+e1/x) = 0/(1+0) = 0
✓ Conclusion
RHD = 1 ≠ 0 = LHD, so f′(0) does not exist — f is not differentiable at x = 0 — even though it is continuous there.
4
Tangent, Normal & Angle of Intersection
Find the angle of intersection of the circles x2 − 4x + y2 = 0 and x2 + y2 = 8.
1
Find Intersection Points
Subtract the second equation from the first to find x.
x2−4x+y2−(x2+y2) = 0−8 ⇒ −4x = −8 ⇒ x = 2
Substituting: 4 + y2 = 8 ⇒ y = ±2
Intersection points: (2, 2) and (2, −2)
2
Find Slopes of Both Circles at (2, 2)
Differentiate each implicitly, then substitute (2, 2).
Circle 1: 2x−4+2y(dy/dx) = 0 ⇒ dy/dx = (4−2x)/(2y)
At (2,2): m1 = (4−4)/(4) = 0

Circle 2: 2x+2y(dy/dx) = 0 ⇒ dy/dx = −x/y
At (2,2): m2 = −2/2 = −1
3
Apply the Angle Formula
Use tan θ = |(m1 − m2)/(1 + m1·m2)| with m1 = 0, m2 = −1.
tan θ = |(0−(−1))/(1+0·(−1))| = |1/1| = 1
θ = tan−1(1) = 45°
✓ Final Answer
The angle of intersection of the two circles at (2, 2) is θ = 45°.
5
Related Rates
Water runs into a conical tank at 9 ft3/min. The tank stands point-down, height 10 ft, base radius 5 ft. How fast is the water level rising when the water is 6 ft deep?
1
Identify Variables and Given Information
Let y = water depth (variable), x = radius of water surface (variable). Given dV/dt = 9 ft3/min; tank: H = 10, R = 5.
dV/dt = 9 ft3/min  |  y = 6 ft (at moment of interest)  |  dy/dt = ?
2
Find the Linking Equation
By similar triangles: x/y = R/H = 5/10 = 1/2, so x = y/2. Write V in terms of y only.
V = (1/3)π x2 y = (1/3)π(y/2)2y = (1/3)π(y2/4)y = πy3/12
3
Differentiate with Respect to Time
Differentiate both sides with respect to t.
dV/dt = (π/12) · 3y2 · (dy/dt) = (π/4) y2 (dy/dt)
4
Substitute Known Values and Solve
Plug in dV/dt = 9 and y = 6.
9 = (π/4)(36)(dy/dt) = 9π · (dy/dt)
dy/dt = 9/(9π) = 1/π ft/min
✓ Final Answer
The water level is rising at dy/dt = 1/π ≈ 0.318 ft/min when the water is 6 ft deep.

Common Pitfalls & Last-Minute Tips

🚫 Limits
⚠ Pitfall 1
Applying L'Hôpital's Rule when the limit is not 0/0 or ∞/∞. L'Hôpital only applies to indeterminate forms. Applying it to something like lim (x2+1)/(x+2) as x→1 — which is already 2/3 — is wrong. Always check whether the form is indeterminate first.
⚠ Pitfall 2
Confusing ε and δ. In the ε-δ proof, ε controls the output (how close f(x) is to L), and δ controls the input (how close x is to c). You find δ in terms of ε — never the other way around.
⚠ Pitfall 3
Assuming a limit doesn't exist just because direct substitution gives 0/0. This is an indeterminate form, not a proof of nonexistence. Always simplify (factor, rationalise, use standard limits) before concluding.
🚫 Continuity & Differentiability
⚠ Pitfall 4
Thinking differentiability implies a corner is impossible. A function is not differentiable at a corner point because the left and right derivatives are unequal. Always check both one-sided derivatives when the formula changes.
⚠ Pitfall 5
Skipping the third continuity condition. It is not enough that f(c) exists and the limit exists separately — they must be equal. Forgetting to verify limx→c f(x) = f(c) is a classic exam error.
🚫 Derivatives & Applications
⚠ Pitfall 6
Confusing a critical point with an extremum. f′(c) = 0 only means c is a critical point; it could still be an inflection point. Always apply the 1st or 2nd derivative test to classify.
⚠ Pitfall 7
Forgetting to check endpoints in absolute extrema problems. On a closed interval [a, b], the absolute maximum or minimum may occur at an endpoint, not at a critical point. Evaluate f at all critical points AND both endpoints.
⚠ Pitfall 8
Incorrect sign convention in related rates. If a quantity is decreasing (e.g. a ladder sliding down), its derivative with respect to t is negative. Always assign signs correctly before substituting.
⚠ Pitfall 9
Applying Rolle's Theorem without checking all three conditions. You must verify: (1) f continuous on [a,b], (2) f differentiable on (a,b), and (3) f(a) = f(b). If any condition fails, the conclusion may not hold.

✅ Last-Minute Tips for Tomorrow

💡 Tip 1 — Standard Limits are Everything
Memorise the six or seven key standard limits (sin θ / θ → 1, (ex−1)/x → 1, log(1+x)/x → 1, (1+1/n)n → e). Most limit problems reduce to one of these after a substitution.
💡 Tip 2 — Always State Your Tools
When using L'Hôpital, Rolle's, or MVT, write the theorem name and verify its conditions before applying it. Examiners often award marks for the setup, not just the answer.
💡 Tip 3 — Differentials for Approximations
For approximation questions, dy = f′(x)·dx. Identify f, a, and dx, compute f(a) + dy. Remember the key linearization: (1+x)k ≈ 1 + kx for small x.
💡 Tip 4 — Related Rates Checklist
Every related rates problem follows the same 6 steps: (1) diagram, (2) name variables, (3) write given rates, (4) write what to find, (5) write linking equation, (6) differentiate w.r.t. t, then substitute.
💡 Tip 5 — Curve Sketching Sign Tables
For curve sketching, always make a sign table for y′ (rising/falling) and y″ (concavity). Arrows in each interval (↗↘ and ∪∩) let you draw the curve with confidence before computing many specific points.
💡 Tip 6 — Oblique Asymptote Shortcut
For oblique (slant) asymptotes on rational functions where deg(numerator) = deg(denominator) + 1: do polynomial long division. The linear quotient is the asymptote; the remainder term vanishes as x → ±∞.
